# 接口文档示例 #
# 根据appkey获取所属工作台信息
### 1、URL
GET /workbench
### 2、参数说明
URL参数 

| 参数名 | 是否必选 | 类型 | 描述 |
| :-----:| :----: | :----: | :----: |
| appkey | 是 | String | 应用的appkey |


### 3、返回结果
```
{
    "code": "success",
    "message": null,
    "data": {
        "id": "535fcfc60d0a11eabd370a580af40444",
        "name": "测试产品创建内部工作台-ISV",
        "type": "ISV",
        "isInternal": true,
        "productId": null,
        "isOwner": false,
        "createdAt": "2019-11-22 17:27:38"
    }
}

```

# 修改工作台名称
### 1、URL
PATCH /workbenches/{id}?name={name}
### 2、参数说明
URL参数

| 参数名 | 是否必选 | 类型 | 描述 |
| :-----:| :----: | :----: | :----: |
| id | 是 | String | 工作台id |
| name | 是 | String | 工作台名称 |



### 3、返回结果
```
{
    "code": "success",
    "message": null,
    "data": {
        "id": "903694f47050485e99913e1295f54f21",

        "isInternal": true,
```
